# SchoolProject
## Install
### Server
1. Установите все зависимости сервера.
>cd <севрер>

>pip install -r requirements.txt
2. Вам нужна уже готовая PostgreSQL:
3. Заполните все конфиги в файле config.yml
4. Запустите main.py
>python main.py

### Client
1. Установите зависимости клиента
> cd <клиент>

> pip install -r requirements.txt
3. Запустите main.py
> python main.py

P.S Рекомедую Python 3.12
